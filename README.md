# Incident Response Investigation Using Wireshark

## Overview

This project demonstrates basic incident response and network traffic analysis using Wireshark.

## Tools Used

* Wireshark
* GitHub

## Project Structure

* screenshots/ - Evidence collected during analysis
* reports/ - Investigation report
* sample-pcap/ - Packet capture files used for analysis

## Analysis Performed

* Protocol Hierarchy Analysis
* Endpoint Analysis
* DNS Traffic Analysis

## Findings

The packet capture contained normal network activity, including DNS queries, local host communication, and standard network protocols. No obvious malicious activity was identified during the investigation.

## Skills Demonstrated

* Packet Analysis
* DNS Analysis
* Endpoint Investigation
* Incident Response Documentation
* Wireshark Traffic Analysis

## Conclusion

This project provided hands-on experience using Wireshark to investigate network traffic and document findings in a structured incident response report.
